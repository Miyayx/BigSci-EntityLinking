##Database
###MySQL
* Fedora:

        yum install MySQL-python

* Ubuntu:

        apt-get install libmysqlclient-dev python-dev
        pip install MySQL-python

###Virtuoso
* Fedora:

        yum install unixODBC unixODBC-devel
        pip install pyodbc 
        pip install rdflib

* Ubuntu:

        apt-get install unixODBC unixODBC-dev
        pip install pyodbc 
        pip install rdflib
